# martinique-beach
site qui predis l affluence des plages de martinique
martinique-beaches/
├── backend/
│   ├── server.js        (Le chef d'orchestre, gère les requêtes)
│   ├── algorithm.js     (Le fichier qui calcule le score d'affluence)
│   ├── amadeus.js       (Futur fichier pour l'API des vols)
│   └── weather.js       (Futur fichier pour la météo)
├── frontend/
│   ├── index.html       (L'interface visuelle)
│   ├── style.css        (Le design)
│   └── app.js           (La logique de la carte Leaflet)
└── .env                 (Fichier caché pour tes clés API secrètes)
